## Directory Layout
Here's the directory structure:

```
.
├── api-docs          - Folder to keep all the local documentation
│   ├── assets        - Folder with all the css/js assets
│   └── ...           - Documentation folders that hold docs for each exchange
│
├── data-feeds        - Folder to keep data scraping scripts in
│   ├── exchng1       - Scraping scripts for an exchange (in this case named exchng1)
│   |   ├── src       - Source Folder for code
│   |   ├── README.md - README for exchange scripts
|   |   └── ...       - Additional files and folders
|   |
│   └── ...           - Additional exchange folders
|
├── README.md         - This doc
│
├── api_docs.md       - Checklist for api documentation process, also contains
│                     links for all the docs main pages
├── .gitignore        - .gitignore file
└── ...

```
